# dapp-kit-react

React hooks and components for building decentralized applications on the [Haneul](https://haneul.io)
blockchain. Built on top of
[`@haneullabs/dapp-kit-core`](https://www.npmjs.com/package/@haneullabs/dapp-kit-core).

Provides a `DAppKitProvider`, ready-to-use `ConnectButton` and `ConnectModal` components, and hooks
like `useDAppKit`, `useCurrentAccount`, `useCurrentWallet`, and more.

## Installation

```sh
npm i @haneullabs/dapp-kit-react @haneullabs/haneul
```

## Documentation

See the [dApp Kit docs](https://sdk.haneul-labs.com/haneul/dapp-kit) for full usage instructions,
guides, and API reference.
