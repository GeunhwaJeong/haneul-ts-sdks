---
title: Haneul TypeScript SDK Quick Start
---

The Haneul TypeScript SDK is a modular library of tools for interacting with the Haneul blockchain. Use it
to send queries to RPC nodes, build and sign transactions, and interact with a Haneul or local network.

## Installation

```sh npm2yarn
npm i @haneullabs/haneul
```

## Network locations

The following table lists the locations for Haneul networks.

| Network | Full node                             | faucet                                   |
| ------- | ------------------------------------- | ---------------------------------------- |
| local   | `http://127.0.0.1:9000` (default)     | `http://127.0.0.1:9123/v2/gas` (default) |
| Devnet  | `https://fullnode.devnet.haneul.io:443`  | `https://faucet.devnet.haneul.io/v2/gas`    |
| Testnet | `https://fullnode.testnet.haneul.io:443` | `https://faucet.testnet.haneul.io/v2/gas`   |
| Mainnet | `https://fullnode.mainnet.haneul.io:443` | `null`                                   |

<Callout type="warn">

Use dedicated nodes/shared services rather than public endpoints for production apps. The public
endpoints maintained by Haneul Labs (`fullnode.<NETWORK>.haneul.io:443`) are rate-limited, and support
only 100 requests per 30 seconds or so. Do not use public endpoints in production applications with
high traffic volume.

You can either run your own Full nodes, or outsource this to a professional infrastructure provider
(preferred for apps that have high traffic). You can find a list of reliable RPC endpoint providers
for Haneul on the [Haneul Dev Portal](https://haneul.io/developers#dev-tools) using the **Node Service** tab.

</Callout>

## Module packages

The SDK contains a set of modular packages that you can use independently or together. Import just
what you need to keep your code light and compact.

- [`@haneullabs/haneul/client`](/haneul/clients) - A client for interacting with Haneul RPC nodes.
- [`@haneullabs/haneul/bcs`](/haneul/bcs) - A BCS builder with pre-defined types for Haneul.
- [`@haneullabs/haneul/transactions`](/haneul/transaction-building/basics) - Utilities for building and
  interacting with transactions.
- [`@haneullabs/haneul/keypairs/*`](/haneul/cryptography/keypairs) - Modular exports for specific KeyPair
  implementations.
- [`@haneullabs/haneul/verify`](/haneul/cryptography/keypairs#verifying-signatures-without-a-key-pair) -
  Methods for verifying transactions and messages.
- [`@haneullabs/haneul/cryptography`](/haneul/cryptography/keypairs) - Shared types and classes for
  cryptography.
- [`@haneullabs/haneul/multisig`](/haneul/cryptography/multisig) - Utilities for working with multisig
  signatures.
- [`@haneullabs/haneul/utils`](/haneul/utils) - Utilities for formatting and parsing various Haneul types.
- [`@haneullabs/haneul/faucet`](/haneul/faucet) - Methods for requesting HANEUL from a faucet.
- [`@haneullabs/haneul/zklogin`](/haneul/zklogin) - Utilities for working with zkLogin.
